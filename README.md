artemisia
=========

Kleines Framework das mittels pandocs markdown eine HTML Präsentation
erstellt.

Zum Einsatz kommen die Programme:

- pandoc
- (g)vim
- make
- imagemagick
- Optional pdflatex um die Präsentation als PDF zu erstellen
